# 🎮 玩家吧 - 游戏社区交流平台

一个基于 React 的游戏社区论坛应用，支持帖子发布、评论互动、用户登录注册等功能。

## ✨ 功能特性

- 📝 **帖子管理** - 发布、浏览、删除帖子，支持多话题分类
- 💬 **评论系统** - 评论、回复、点赞功能
- 👤 **用户系统** - 登录、注册、个人头像设置
- 🔐 **管理后台** - 用户管理、帖子管理、话题分类管理
- 📱 **响应式设计** - 适配桌面端和移动端
- 💾 **本地存储** - 数据保存在浏览器 localStorage，无需后端

## 🛠️ 技术栈

| 类型 | 技术 |
|------|------|
| 前端框架 | React 18 |
| 路由管理 | React Router 6 |
| CSS 框架 | Tailwind CSS |
| 构建工具 | Vite 5 |
| 字体 | Google Fonts (Noto Sans SC) |

## 🚀 如何使用

### 在线演示
直接双击 `玩家吧-作品集.html` 文件，在浏览器中打开即可使用。

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```


## 🎯 页面说明

| 页面 | 路由 | 功能 |
|------|------|------|
| 首页 | `#/` | 帖子列表、话题分类、搜索 |
| 帖子详情 | `#/post/:id` | 查看帖子、评论互动 |
| 登录/注册 | `#/login` | 用户认证 |
| 管理后台 | `#/admin` | 用户管理、帖子管理（需管理员权限）|

## 💡 技术亮点

1. **单文件部署** - 使用 `vite-plugin-singlefile` 将所有资源内联到单个 HTML 文件，可直接双击运行
2. **HashRouter 路由** - 支持 `file://` 协议下的前端路由
3. **模块化设计** - 组件化开发，代码结构清晰
4. **无后端依赖** - 所有数据存储在 localStorage，适合演示和面试展示
